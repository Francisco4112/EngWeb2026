## Resumo
Este trabalho cria um `json-server` com o dataset de reparações e um servidor
aplicacional em Node.js que disponibiliza três rotas HTML: listagem completa de
reparações, agregação dos tipos de intervenção com o número de ocorrências e
agregação dos modelos de viatura com o total de reparações por modelo(de notar que 
neste dataset todos os modelos que aparecem só foram reparados uma vez).
